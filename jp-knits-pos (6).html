🔄 Multi-Device Real-Time Sync
Powered by Firebase Firestore — Free tier: 50,000 reads + 20,000 writes/day. More than enough for a shop.
🔴 Not connected — orders save locally only (this device)
🚀 Setup in 5 minutes (free):
1. Go to console.firebase.google.com → Create a project (e.g. "jpknits-pos")
2. Click Firestore Database → Create database → Start in test mode
3. Go to Project Settings → Your apps → click </> Web → Register app
4. Copy the firebaseConfig values and paste them below
5. Click Connect — all devices with the same config share the same live data ✓
