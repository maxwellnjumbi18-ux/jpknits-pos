📲 Install JP Knits POS as an App

Follow these steps to host your POS online and install it on any phone or computer — completely free.

1

Create a free GitHub account

Go to github.com and sign up. It's completely free and takes 2 minutes.

2

Create a new repository

After signing in:

  • Click the + button (top right) → New repository
  • Name it: jpknits-pos
  • Set it to Public
  • Tick Add a README file
  • Click Create repository
3

Upload all these files

Click Add file → Upload files and drag ALL of these files in:

📄 jp-knits-pos.html
📄 manifest.json
📄 sw.js
🖼️ icon-192.png
🖼️ icon-512.png

Then click Commit changes (green button at the bottom).

4

Turn on GitHub Pages

  • Go to your repository → click Settings (top menu)
  • In the left sidebar click Pages
  • Under "Source" select Deploy from a branch
  • Branch: main, folder: / (root)
  • Click Save
⏳ Wait 2–3 minutes, then your POS will be live at:
https://YOUR-USERNAME.github.io/jpknits-pos/jp-knits-pos.html
(replace YOUR-USERNAME with your GitHub username)
5

Install on Android phone

  • Open the URL above in Chrome on your Android phone
  • A banner will appear at the bottom saying "Install JP Knits POS"
  • Tap Install App
  • The app appears on your home screen and opens fullscreen like a native app ✅
6

Install on iPhone / iPad

  • Open the URL in Safari (must be Safari, not Chrome)
  • Tap the Share button (the square with an arrow at the bottom)
  • Scroll down and tap "Add to Home Screen"
  • Tap Add (top right)
7

Install on Windows / Mac computer

  • Open the URL in Chrome or Edge
  • Look for the install icon in the address bar (looks like a monitor with a + sign)
  • Click it → Install
  • The POS opens in its own window without browser toolbars ✅
8

Share with other devices in the shop

Give the same URL to every phone or computer in the shop. Each device can install the app independently. For real-time sync between all devices, configure Firebase in Settings → Multi-Device Sync inside the app.

✅ The URL never changes. Whenever you update the files on GitHub, all installed apps will automatically get the new version.
